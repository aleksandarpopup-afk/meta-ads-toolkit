# Meta Ads Toolkit

Profesionalni alati za Meta Ads performance marketing.

## Moduli
- 📊 Campaign Health Check
- 💰 Budget Pace Kalkulator
- ✍️ Ad Copy Generator
- 🎯 Audience Planner
- 📈 ROAS & Break-Even Kalkulator
- ✅ Launch Checklist (Pixel, CAPI, Events)

## Deploy na Vercel (5 minuta)

### Korak 1 – GitHub
1. Idi na [github.com](https://github.com) i napravi nalog
2. Klikni **New repository** → naziv: `meta-ads-toolkit` → Create
3. Upload-uj sve fajlove iz ovog foldera (drag & drop)

### Korak 2 – Vercel
1. Idi na [vercel.com](https://vercel.com) → **Sign up with GitHub**
2. Klikni **New Project** → izaberi `meta-ads-toolkit` repository
3. Klikni **Deploy** (sve ostalo ostavi default)
4. Za ~1 minutu dobiješ live link!

### Korak 3 – Custom domen (opciono)
1. Kupi domen na [namecheap.com](https://namecheap.com)
2. U Vercel dashboard → Settings → Domains → dodaj domen
3. Namecheap → DNS → dodaj CNAME record koji Vercel pokaže

## Kako ažurirati app

Kada dobiješ novu verziju `App.jsx` fajla:
1. Idi na GitHub repository
2. Klikni na `src/App.jsx` → Edit (olovka ikona)
3. Zameni sadržaj novim kodom → **Commit changes**
4. Vercel automatski deploya novu verziju za ~30 sekundi ✅
